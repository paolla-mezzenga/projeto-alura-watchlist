# 🌸 Watchie - Minha Watchlist de Séries 🌸

## ✨ Sobre o Projeto

O projeto começou como uma ideia de criar um "e-commerce" inspirado no AluraGeek, mas eu não gostei muito da ideia. Então, com muita criatividade, decidi transformá-lo em uma **Watchlist de Séries**! 📺💖 Agora, em vez de vender produtos, o objetivo é adicionar, listar e excluir suas séries favoritas de forma super simples e divertida! 🌈

Usei **HTML**, **CSS**, **JavaScript** e o **JSON Server** para criar uma API fake e armazenar as séries. 

## 💖 Funcionalidades

- **Adicionar série**: Preencha o formulário com o nome, gênero, número de temporadas e capa da série. ✍️
- **Listar séries**: Veja as séries que você adicionou em cards super fofos! 🎬💫
- **Excluir série**: Remova qualquer série da sua lista quando quiser! 🗑️
